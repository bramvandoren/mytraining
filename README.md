# ⚽ Football Training Builder

A modern web application for football (soccer) coaches to quickly create, manage, and visualize training sessions in just a few clicks.

---

## 🚀 Features

### 📚 Exercise Library

* Browse a collection of training exercises
* Filter by:

  * Age group (U6, U8, U10, …)
  * Skill level (beginner, intermediate, advanced)
  * Duration
  * Field size
  * Type (technique, passing, finishing, game forms, warm-up)
* Search exercises by name
* Visual previews (field diagrams)
* Support for images and videos

---

### 🧠 Training Builder

* Select multiple exercises
* Drag & drop to reorder
* Automatic total duration calculation
* Add notes per exercise

---

### 💾 Save & Manage Trainings

* Save training sessions with:

  * Name
  * Date
  * Age category
* View and reuse previous trainings

---

### 📱 Matchday View

* Mobile-friendly interface
* Clear step-by-step training overview
* Designed for use on the field

---

### 🎨 Custom Exercise Builder

* Create your own exercises
* Interactive 2D football field editor:

  * Add cones, goals, players
  * Draw arrows (movement/passing)
  * Define zones
* Save as reusable exercises
* Stored as JSON + preview image

---

### 🖼️ Media Support

* Upload exercise images (field diagrams)
* Upload short training videos
* Stored using Supabase Storage

---

## 🛠️ Tech Stack

* **Frontend:** React / Next.js
* **Backend:** Supabase
* **Database:** PostgreSQL (via Supabase)
* **Storage:** Supabase Storage
* **Deployment:** Vercel
